# 📊 Sales Performance Dashboard

## Overview
This project is an Excel-based dashboard that analyzes sales performance across products, cities, and payment methods. It provides key insights into revenue distribution and customer purchasing behavior.

- UPI: ₹434,500  
- Card: ₹250,000

## Tools Used
- Microsoft Excel (Pivot Tables, Charts, Dashboard Design)

## Insights
- **Laptops** generate the highest revenue.  
- **Delhi** leads in city-wise sales.  
- **UPI payments** dominate overall transactions.
